# MEFAI Browser

[![Build](https://github.com/mefai-dev/mefai-browser/actions/workflows/build.yml/badge.svg)](https://github.com/mefai-dev/mefai-browser/actions/workflows/build.yml)

[![Platform](https://img.shields.io/badge/platform-win%20%7C%20mac%20%7C%20linux-lightgrey.svg)]()
[![License: Apache 2.0](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](LICENSE)

AI native trading browser built on Electron. Integrates directly with MEFAI Engine for a unified trading workspace.

## Getting Started

```bash
npm install
npm start
```

## Building

Platform specific builds:

```bash
npm run build:win
npm run build:mac
npm run build:linux
```

Output goes to `dist/`.
